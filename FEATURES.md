# Feature Roadmap

Planned capabilities not yet covered by Scenic. Check off items as they ship.

## Assets

- [ ] Search for assets by name, type, or label
- [ ] Import assets into the project
- [ ] Move, rename, copy, and delete assets
- [ ] Read asset metadata (import settings, dependencies)

## Materials & Shaders

- [x] Create and inspect materials
- [x] Assign shaders to materials
- [x] Get and set material properties (colors, textures, floats, vectors)

## Animation

- [ ] Manage Animator controllers (create, inspect, modify)
- [ ] Create and assign animation clips
- [ ] Configure state machines, transitions, and parameters

## Physics Settings

- [ ] Read and modify global physics settings (gravity, fixed timestep, collision matrix)
- [ ] Perform raycasts and overlap queries

## UI

- [ ] Create and manipulate Canvas UI elements
- [ ] Create and manipulate UI Toolkit elements

## Audio

- [ ] Manage audio mixers and mixer groups
- [ ] Assign and configure audio clips beyond component fields

## Lighting & Rendering

- [ ] Bake lighting
- [ ] Modify render pipeline settings
- [ ] Configure environment and skybox settings

## Build Pipeline

- [ ] Trigger builds from the command line
- [ ] Set build targets and platforms
- [ ] Manage build settings and player settings

## Project Settings

- [ ] Read and modify quality settings
- [ ] Read and modify input mappings
- [ ] Read and modify time settings
- [ ] Read and modify physics settings

## Navigation

- [ ] Bake NavMeshes

## Terrain

- [ ] Create terrain assets

- [ ] Paint details and trees
